# 프로젝트

퀀트는 현재까지 다음의 프로젝트를 개발했습니다.

- 얽힘 라이브러리(EntanglementLib)
    - 각 타입에 안전하고 효율적으로 대응하기 위한 유틸리티, 양자-내성 암호화(Post-Quantum Cryptography, PQC) 알고리즘을 사용한 암호화 도구 및 인증서 발급, TLSv1.3(1.2)
      연결을 위한 매니지먼트를 제공합니다.
- 블루-브릿지(Blue-bridge)
    - 얽힘 라이브러리를 사용한 전자 서명과, ORM 기술이 탑재된 데이터베이스 관리 멀티 모듈 프로젝트입니다.
- 포세이돈(Poseidon)
    - 포세이돈은 온디바이스 TinyML을 활용한 QRAME 프로토콜의 내부 보안 AI 모델입니다.
- 클레임 프로토콜(Quantum-Resilient Adaptive Mesh E2EE Procotol)
    - PQXDH(Post-Quantum Extended Diffie-Hellman) 프로토콜을 기반으로 한 새로운 종단간 암호화(End-To-End Encryption, E2EE) 메신저 시스템입니다.

개별 프로젝트에 대한 세부 개념을 확인하여 자유롭게 개인의 서비스를 구축할 수 있습니다.
