# Projects

Quant has developed the following projects so far.

## EntanglementLib

EntanglementLib is a high-performance security and utility library designed based on Post-Quantum Cryptography (PQC) technology. This library aims to provide the highest level of security and system stability in response to the evolving digital threat environment.

## Blue-bridge

It is a database management multi-module project equipped with digital signatures using EntanglementLib and ORM technology.

## Poseidon

Poseidon is an internal security AI model of the QRAME protocol utilizing on-device TinyML.

## QRAME Protocol (Quantum-Resilient Adaptive Mesh E2EE Protocol)

It is a new End-To-End Encryption (E2EE) messenger system based on the PQXDH (Post-Quantum Extended Diffie-Hellman) protocol.
