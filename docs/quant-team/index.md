# 퀀트 개발자 기초

퀀트 팀의 개발자들은 안전한 기반, 효율적인 구현으로 설계된 프로젝트의 보안성과 외부로부터 개별 소스 산출물의 유연한 접근 방식을 우선시합니다.

퀀트는 기본적으로 당신의 개발 능력에 대해 깊이 평가하려 들지 않습니다. 물론 기초적인 개념은 알고 있다는 전재를 두지만요. 중요한 것은 안전함과 익명성을 생각하는 사고 방식과 순수 개발에 대한 열정, 의욕이며 이
강점을 보유하고만 있다면 그것만으로 함께 달려나갈 수 있는 길을 만들어낼 수 있기 때문입니다. 퀀트의 개발자라면 자신의 실력에 대해 좌절하거나 한탄하지 마시길 바랍니다.

당신이 퀀트의 개발자라면 기본적으로 다음의 방법과 단계를 통해 효율적인 개발을 수행하실 수 있습니다.
